# Sistema-de-Gestion-de-Tareas-y-Proyectos-con-rbol-Binario-de-Busqueda-BST
Aplicación de escritorio desarrollada en Python con Tkinter que utiliza un Árbol Binario de Búsqueda (BST) para gestionar tareas y proyectos. Permite insertar, buscar y eliminar actividades de forma eficiente, aprovechando la organización y rapidez del BST frente a estructuras lineales como listas o arreglos.
